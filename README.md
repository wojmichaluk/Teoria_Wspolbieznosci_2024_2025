# TW_2024_2025

### Repository for TW (Teoria Współbieżności - Theory of Concurrency) course at AGH University

Repozytorium zawiera moje rozwiązania zadań domowych z Teorii Współbieżności - katalogi `zd1`, `zd2` i `zd3`.
Każdy z tych katalogów zawiera polecenie zadania (plik **zd*i*_polecenie.pdf**, `i = 1, 2, 3`) oraz raport z zadania - pliki odpowiednio `tw_zd1.pdf`, `Opis_programu.pdf` i `tw_zd3.pdf`. 
Poza tym są tam też właściwe rozwiązania (programy) - struktura plików jest opisana w raporcie do każdego zadania.
Rozwiązanie 1. zadania jest w Javie + do wykresów Python i Jupyter Notebook, rozwiązania 2. i 3. zadania są już w samym Pythonie.

Wszystkie zadania zostały ocenione na 2/2 punkty, chociaż w 3. zadaniu pojawiają się istotne błędy (możliwe, że nie z mojej winy, ale kto wie).
Przede wszystkim mam nadzieję, że zamieszczone tu rozwiązania okażą się przydatne. Korzystajcie rozważnie!
